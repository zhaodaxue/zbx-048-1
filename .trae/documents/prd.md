## 1. 产品概述

日偏食观测阶段倒计时屏——为现场观众设计的大屏站立观看界面，实时显示日偏食四个关键阶段（初亏、食甚、复圆、结束）的倒计时，帮助观众把握观测节奏与安全提醒。

- 目标用户：日偏食观测现场的站立观众
- 核心价值：让观众一目了然地知道距下一阶段还剩多久，并在户外裸眼模式下提供安全警示

## 2. 核心功能

### 2.1 功能模块

1. **倒计时主屏**：顶部超大倒计时数字，每秒刷新显示距下一阶段的剩余分秒
2. **阶段进度条**：横向阶段条标示当前所处区间，四个阶段节点可视化
3. **模式切换**：室内投影 / 户外裸眼模式切换，户外模式下食甚前后 3 分钟标红并显示"禁止直视"警告
4. **科普弹窗**：点击阶段名弹出该阶段一句科普说明（硬编码文案）

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 倒计时主屏 | 倒计时区域 | 顶部大字显示距下一阶段剩余分秒，每秒刷新 |
| 倒计时主屏 | 阶段进度条 | 横向四段式进度条，当前阶段高亮，已完成阶段置灰，未来阶段半透明 |
| 倒计时主屏 | 模式切换 | 右上角切换按钮，室内/户外模式，影响食甚区域显示风格 |
| 倒计时主屏 | 安全警告 | 户外模式下食甚前后3分钟区域标红+闪烁"禁止直视" |
| 倒计时主屏 | 科普弹窗 | 点击阶段名弹出遮罩层+卡片，显示一句科普说明 |

## 3. 核心流程

用户打开页面 → 看到当前阶段与倒计时 → 观察阶段进度条了解全局 → 切换户外模式获取安全提示 → 点击阶段名了解科普知识

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深空宇宙风——深邃暗色背景，星辰点缀，天文观测氛围
- **主色**：深靛蓝 `#0a0e27`（背景）、金色 `#f0c040`（倒计时高亮）、冷白 `#e8eaf6`（正文）
- **辅助色**：暗紫 `#1a1a3e`（阶段条底色）、赤红 `#ff2d2d`（户外警告）、淡青 `#64ffda`（当前阶段标记）
- **字体**：倒计时数字使用等宽数字字体（JetBrains Mono / Orbitron），阶段名与正文使用思源黑体
- **按钮风格**：圆角胶囊形，半透明玻璃质感
- **布局**：全屏垂直居中，倒计时占视觉重心 60%，阶段条居中 80% 宽度

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 倒计时主屏 | 倒计时区域 | 超大等宽数字（6rem+），金色发光效果，底部显示目标阶段名 |
| 倒计时主屏 | 阶段进度条 | 横向四段渐变条，节点圆点+阶段名标签，当前阶段脉动动画 |
| 倒计时主屏 | 模式切换 | 右上角胶囊按钮，图标+文字，切换时过渡动画 |
| 倒计时主屏 | 安全警告 | 红色脉冲背景条，白色粗体"禁止直视"闪烁文字 |
| 倒计时主屏 | 科普弹窗 | 半透明遮罩层，居中卡片，阶段名标题+一句话说明，点击遮罩关闭 |

### 4.3 响应式设计

- 桌面优先（大屏站立观看场景），最小宽度 1280px
- 1920×1080 全屏最佳
- 文字与元素使用 rem/vw 单位适配不同分辨率

## 5. 阶段数据

内置一次日偏食事件四阶段时刻（示例）：

| 阶段 | 时刻 | 科普说明 |
|------|------|---------|
| 初亏 | 08:26 | 月球边缘刚接触太阳圆面，日偏食开始，太阳出现微小缺口 |
| 食甚 | 09:43 | 月球遮住太阳面积最大时刻，此时日面缺损最显著 |
| 复圆 | 11:03 | 月球完全离开太阳圆面，日偏食结束，太阳恢复圆形 |
| 结束 | 11:15 | 整个日偏食观测活动结束，感谢观测 |
